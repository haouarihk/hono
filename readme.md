# Server for EnvayaSMS



## Getting Started

## Docker
First build the bunjs image
```sh
docker build -t bunjs:v6.11 ./bunjs
```
run the docker compose file
```sh
docker-compose up -d
```

### Cloning the repo

```sh
bun create hono ./NAME_HERE
```

### Development

```
bun run start
```

Open http://localhost:3000 with your browser to see the result.
